# CI Design

**Date:** 2026-06-18  
**Status:** Approved — pending implementation  
**Scope:** GitHub Actions CI — Bun test suite + compiled binary E2E

---

## Goal

Every push and PR gets a green/red signal on two independent concerns: does the test suite pass in dev mode, and does the compiled binary work end-to-end with its WASM sidecars.

---

## Workflow: `.github/workflows/ci.yml`

Single workflow file, two jobs.

**Trigger:**
```yaml
on:
  push:
    branches: ["**"]
  pull_request:
    branches: [main]
```

---

## Job 1 — `test`

**Runner:** `ubuntu-latest`  
**Purpose:** Validate dev-mode correctness — typecheck, tests, lint.

Steps:
1. `actions/checkout@v4`
2. `oven-sh/setup-bun@v2` — latest stable Bun 1.x
3. `bun install --frozen-lockfile`
4. `bun run typecheck`
5. `bun test`
6. `bunx @biomejs/biome check .`

All four commands must pass. Any failure fails the job.

---

## Job 2 — `binary`

**Runner:** `ubuntu-latest`  
**Depends on:** `test` (skips if test fails — no point building a broken binary)  
**Purpose:** Validate the compiled binary distribution with real WASM sidecars.

Steps:
1. `actions/checkout@v4`
2. `oven-sh/setup-bun@v2`
3. `bun install --frozen-lockfile`
4. `bun run build` → produces `dist/tsgraph` (Linux ELF binary)
5. Copy WASM sidecars into `dist/`:
   - `node_modules/web-tree-sitter/web-tree-sitter.wasm → dist/`
   - `node_modules/tree-sitter-typescript/tree-sitter-typescript.wasm → dist/`
6. E2E test sequence (bash, `set -euo pipefail`):
   ```bash
   ./dist/tsgraph build ./fixtures/simple-ts
   ./dist/tsgraph query greet --db ./fixtures/simple-ts/.tsgraph.db > /tmp/query-out.json
   grep -q '"name"' /tmp/query-out.json
   grep -q 'greet' /tmp/query-out.json
   ./dist/tsgraph explain-edges > /tmp/edges-out.json
   grep -q 'edge_kinds' /tmp/edges-out.json
   ```
7. Cleanup: `rm -f fixtures/simple-ts/.tsgraph.db*`

---

## What This Does Not Cover

- Node fallback bundle (`bun build:node`) — Phase 3, deferred
- Release / publish workflow — not yet designed
- Matrix builds (multiple OS or Bun versions) — not needed yet
- Performance benchmarks — out of scope
